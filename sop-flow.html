SOP Flow

Collapsible steps; anchor links let you focus any node.

  • 1) Intake #
    • 1.1) Triage Request #

      Check scope, urgency, owner, due date. Create ticket.

    • 1.2) Gather Docs #

      Collect SOPs, templates, prior work. Store in canonical folder.

  • 2) Execute #
    • 2.1) Draft #

      Work in checklist order; log decisions and open questions.

    • 2.2) Review #

      Peer review for accuracy, compliance, and dependencies.

  • 3) Close-Out #
    • 3.1) Handbook Update #

      Update SOP library and cross-link to training materials.

    • 3.2) Retro / Metrics #

      Capture time, blockers, and improvements; feed into next cycle.
